Lidar mapping using Lidar lite v3 module from Garmin. It can produce a map with accuracy of + or - 3cm. The SMSF is on March 5 at the San Mateo events center. feel free to attend.

Minimax

Made a bot to play tic tac toe which is unbeatable by using a minimax algorithm. This uses a decision tree which traces all possible outcomes. From here it will score each and select the best. To do this it predicts what is best for itself and plays against what is best for the player. After it reports the best score based on if it thinks it will lose or win.

Chatbot using machine Learning

Trained a Chatbot through 1000 epochs with an intent understanding using a neural net composed of inputs all connected to two rows of 8 predictive nodes which finally connect to an output node with the probabilities of the top 5 answers. from there I choose the top one if it meets a threshold and reject if it does not.

Facial Recognition

Created a facial recognition python package leveraging Faiss, an algorithm made by Facebook. This used a 128-bit vector encoding of color shifts and their direction to compare to a video stream. With Faiss, the comparison process was made quicker as it could group similar encodings and use a tree to navigate to the most accurate one.

Water Purifier

Using the power of a vacuum at 27-30HG water boils and with this principle, I would have two containers one with dirty water and a large empty chamber for condensing. Once a vacuum was pulled for a bit it could be released and the vaporized water would then condense in the large chamber as clean water.
